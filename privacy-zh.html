隐私政策

最后更新日期：05-01-2026

欢迎使用 Spicy Library。 我们尊重您的隐私，并致力于保护您的数据安全。

1. 我们收集的信息

  • 通过 Pi Network 获取的用户名和 UID
  • 购买记录
  • 作者销售数据
  • 用于提现申请的 Pi 钱包地址

我们不会收集： 密码、私钥或其他敏感个人信息。

2. 信息的使用方式

我们仅将您的数据用于：

  • 应用程序功能和登录验证
  • 处理付款和购买交易
  • 交付电子书内容
  • 计算收益并处理提现请求

3. 数据保护

数据存储于 Firebase Firestore， 所有付款均通过 Pi Network 官方 API 处理。

4. 数据共享

除法律要求外， 我们不会向第三方出售或共享用户数据。

5. 用户责任

用户有责任：

  • 保护自己的 Pi Network 账户安全
  • 不得上传侵犯版权或知识产权的内容

6. 更新

本隐私政策可能随时更新， 所有变更都会发布在本页面。

7. 联系方式

如有任何问题：
