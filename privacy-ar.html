سياسة الخصوصية

آخر تحديث: 05-01-2026

مرحباً بك في Spicy Library. نحن نحترم خصوصيتك ونلتزم بحماية بياناتك.

1. المعلومات التي نجمعها

  • اسم المستخدم ومعرف المستخدم (UID) عبر Pi Network
  • سجل المشتريات
  • بيانات المبيعات الخاصة بالمؤلفين
  • عنوان محفظة Pi لطلبات السحب

نحن لا نجمع: كلمات المرور أو المفاتيح الخاصة أو أي معلومات شخصية حساسة أخرى.

2. كيفية استخدام المعلومات

نستخدم بياناتك فقط من أجل:

  • تشغيل التطبيق وتسجيل الدخول
  • معالجة المدفوعات وعمليات الشراء
  • تسليم الكتب الرقمية للمستخدمين
  • حساب الأرباح ومعالجة طلبات السحب

3. حماية البيانات

يتم تخزين البيانات باستخدام Firebase Firestore، كما تتم معالجة جميع المدفوعات عبر واجهات Pi Network الرسمية.

4. مشاركة البيانات

لا نقوم ببيع أو مشاركة بيانات المستخدمين مع أي طرف ثالث، إلا إذا كان ذلك مطلوباً بموجب القانون.

5. مسؤولية المستخدم

يتحمل المستخدم المسؤولية عن:

  • حماية حسابه على Pi Network
  • عدم رفع محتوى ينتهك حقوق النشر أو الملكية الفكرية

6. التحديثات

قد يتم تحديث سياسة الخصوصية هذه في أي وقت، وسيتم نشر أي تغييرات على هذه الصفحة.

7. التواصل
