🧾服务说明

Spicy Library 是一个使用 Pi Network 货币购买和出售电子书的数字平台。

2

👤账户

  • 仅可通过 Pi Network 登录
  • 用户对其账户上的所有活动负责
3

©️内容与版权

  • 作者只能上传其拥有版权或已获得合法分发许可的内容。
  • 上传内容即表示作者确认拥有必要的权利。
  • 如收到版权侵权举报，被举报内容在审核期间可能会被删除或禁用。
  • 用户可通过发送邮件至 spicylibrarysupport@gmail.com 举报版权侵权行为。
  • 屡次上传侵权内容的账户可能会被暂停或永久删除。
4
⚠ 零容忍

🚫禁止内容（性 / 色情材料）

严禁在 Spicy Library 上传任何形式的性、色情或露骨性内容——包括文字、图片、书籍封面或任何其他材料。

  • 任何包含性、色情或露骨性内容的图书提交将不会获得批准，并将在审核过程中被拒绝。
  • 若在批准后发现此类内容，该图书将立即从平台上移除。
  • 任何以性方式描绘或涉及未成年人的内容都是绝对禁止的，除永久删除账户外，还将上报至相关部门。
  • 屡次尝试上传禁止的性或露骨内容的账户可能会被暂停或永久封禁。
  • "Spicy Library" 这一名称仅指代广泛多样的图书合集，并不暗示或允许成人、性或露骨内容。
5

💳购买与支付

  • 支付仅使用 Pi 货币进行
  • 所有购买均为最终交易，不可退款
6

💰收益与提现

  • 作者获得销售价格的 70%
  • 最低提现金额为 5 Pi
  • 大多数情况下提现会即时处理。但在特殊情况或技术问题下，处理可能需要 1-2 个工作日。

⭐精选徽章

  • "精选"徽章是平台可能授予某些精选图书的一种可选标识。
  • 该徽章并不代表所有权、认可或任何保证的质量水平——它仅由平台自行决定用于突出显示部分图书。
  • 精选徽章是临时性的，并非永久有效。平台可随时为任何图书添加、更改或移除该徽章，恕不另行通知。
  • Spicy Library 保留随时自行决定并出于任何原因移除任何图书精选徽章的权利。
🗑️
⚠ 不可撤销

🗑️删除图书

  • 作者可随时在"我的书籍"部分删除自己上传的图书。
  • 删除图书是永久且不可撤销的操作。一旦删除，该图书、其封面、文件及列表信息将无法恢复。
  • 如果被删除的图书仍有未提现的收益（尚未申请提现的收益），这些收益将被永久损失，之后无法恢复或追讨。
  • 强烈建议作者在删除图书之前先申请提现任何可用收益。
  • 删除图书不会影响读者在删除前已购买的副本——这些读者仍可在"我的购买"中访问其已购买的副本。

📩举报版权侵权

如果您认为 Spicy Library 上的内容侵犯了您的版权，请发送邮件至 spicylibrarysupport@gmail.com，并包含以下信息：

  • 您的姓名及联系方式
  • 受版权保护作品的描述
  • 被举报内容的链接或标题
  • 声明您认为该使用未经授权

我们将审核所有举报，并可能在适当情况下删除或禁用对被举报内容的访问。
